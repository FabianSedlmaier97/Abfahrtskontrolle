# Abfahrtskontrolle
In diesem Repository befindet sich die Web-Oberfläche um einfach und mobil mit dem Webbrowser des Smartphones die Abfahrtskontrolle als LKW-Fahrer durchzuführen und auch zu dokumentieren.

## Änderungen 10.12.2025
- Style.css und Effects.js wurde hinzugefügt. Der Style wurde aus index.html in Style.css ausgelagert. App-Typische verhalten wurde in Effects.js eingefügt.
- History.html wurde hinzugefügt. Hier soll anhand eines Array im Browser die letzten Abfahrtskontrollen festgehalten werden. Angezeigt werden soll im Log lediglich das Datum, Uhrzeit und Kennzeichen (Zugfahrtzeug und Anhänger)
- Backup.html wurde hinzugefügt. Wenn Stammdaten.html und Kontrolle.html soweit fertig gestellt ist, soll das Backup.html ausgelegt werden.
